Bill Miller Photography, New York, NY

Bill Miller Photography shoots building portraits and interiors for New York City building owners and developers.

Wireframe Comp

I presented the idea of using Flickr to organize Mr. Miller’s photographs and to create standard sizes to use to layout the website.

The idea for the layout was to use Flickr to organize Mr. Miller’s photographs and to create standard sizes to work with. The website layout included a JavaScript slideshow and freestanding pop-up modal pages with navigation to view each image.

Bill Miller Photography Comp Wireframe

This initial comp, created in Adobe Illustrator, was used to sell Mr. Miller on the concept.

Only a few client images were used to ensure that Mr. Miller understood this was only a working comp.

Bill Miller Photography Screenshot Wireframe

Screenshots of working web pages is a great way to confer with clients.

Bill Miller’s mark was the angel sculpture from a building shoot. The website colors were chosen to complement his brand.
