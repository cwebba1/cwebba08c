Bill Miller Photography

Real Estate Photography

NYC-based Real Estate photographer Bill Miller Photography needed a new website portfolio including new marketing copy to build SEO and differentiate his services within search-space.

Website Architecture, User Experience and Design

Craig Webb Art proposed a website architecture with multiple photo-galleries. Featuring pop-up JavaScript web pages that transport the viewer into a quiet space where each photograph can be savored, each gallery focuses on specific photography services.

Built with XHTML and JavaScript, the website is designed as a “living portfolio” so that pages can be easily updated or added and new photographs can be added or switched out.

Part of the user interface design intended to enhance use and flow by the photographer is that the website image sizes are standard Flickr.com images sizes.
