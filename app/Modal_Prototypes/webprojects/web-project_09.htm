xVenturesGroup, LLC HandOff

HandOff in Southampton NY offers auto body painting, auto collision repair, and classic car restoration.

xVenturesGroup needed a website and other marketing material to test the market for HandOff, a health-care related web app.

Website Design

Craig Webb Art proposed a website architecture and design, and built HandOffLabs.com, a responsive micro-website for market testing.

Content Development

Craig Webb wrote original content for the website and presentations. Craig Webb created an original brand style and graphic art elements.

User Experience

Craig Webb built UX wireframes and personas to test market viability for a new web application. Craig Webb wrote content for video storyboards.

Front-End Development

Mobile-first responsive website using SASS, JavaScript and Bootstrap 4.

Previous projects include logo design for ZebraFood.com, an online app.
