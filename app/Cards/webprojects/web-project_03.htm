Icarus Group Ironwork

Icarus Group Ironwork provides ornamental Ironwork, Steel Fabrication and Welding Services.

Icarus Group Ironwork needed a website to display the wide range of custom products that they manufacture.

Website Design

Craig Webb Art designed and built the original Icarus Group Ironworks website in 2010 including writing original content. The website has been updated recently to be mobile-first and responsive.

User Experience

The new website is built using current development best practices. Craig Webb sketched layouts and built wireframes to plan development; and built modular prototypes to test UI functions.

Front-End Development

Static website using SASS, JavaScript and Bootstrap 4.
